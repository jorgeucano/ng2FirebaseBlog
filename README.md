# ng2FirebaseBlog

The ng2FirebaseBlog is a GNU project.

For up this project you need npm & angular-cli

Change "defaultFirebase('https://ng2firebaseblog.firebaseio.com/')" in src/main.ts to your firebase url

in terminar type
- npm install
- ng build
- ng serve
